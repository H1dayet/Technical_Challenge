# Saucedemo Login Tests

Playwright tests for Saucedemo login functionality.

## Test Scenarios

1. Valid login - checks redirect to products page
2. Invalid credentials - verifies error message  
3. Empty fields - validates required field errors

## Test Coverage

| Scenario | Validation |
|----------|------------|
| Valid Login | URL redirection + Products page visibility |
| Invalid Login | Error message validation |
| Empty Fields | Required field error message |
